# Data Quality Control & Measurement Analysis Pipeline

This repository contains a Python notebook implementing a simple data processing pipeline:
- Name quality control
- Statistical analysis of biological measurements
- Aggregation of results into a final report

The notebook was originally developed and published on Kaggle.

## Structure
- `data-qc-and-statistical-analysis-pipeline-python.ipynb` : main notebook implementing the QC and measurement analysis pipeline

## Technologies
- Python
- Jupyter Notebook
